The AI-Powered Automated Sentiment Analysis Tool Project

Project SenTagalog is a work-in-progress research project by ADNU undergraduates Kevin Kyle Hendrick Baldo, Ian Raymund Manaog and Levi John Sta. Ana. The goal of the project is to build an automated and Artificial Intelligence powered general-scope sentiment analyzer for Filipino-English tweets labelled neutral, negative, positive and sarcasm using modern techniques used in the field such as Bidirectional Long Short-Term Memory Recurrent Neural Networks (Bi-LSTM RNN), Gated Recurrent Unit Neural Networks (GRU-RNN) and Feedforward Mutilayer Perceptrons (FF MLP) and by observing which model performs the best with the given dataset.
